# seda_sch_icc

Data file of between-school ICC estimates to accompany:

Shear, B. R., Taylor, J., & Fahle, E. M. (2025). Determinants of between-school variation in student achievement: Results from US population data.

Please use the above citation if you use these estimates in your work.

Files:

data-file-all-iccs-rounded.csv: CSV data file containing all 6,021 between-school ICC estimates analyzed in the paper. (Note: sample sizes rounded to the nearest 10 to comply with NCES disclosure review.)

data-file-all-iccs-rounded-codebook.csv: CSV file with variable descriptions.

NOTE: Detailed description of the methods used to produce ICC estimates and interpretation of results are provided in the paper. For questions or additional information please contact Ben Shear (benjamin.shear@colorado.edu).
